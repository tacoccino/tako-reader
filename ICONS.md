# Tako Reader — Icon Manifest
#
# Place all icons in:  icons/  (next to tako_reader.py)
# Format:              PNG, recommended size 32x32 or 16x16
# All icons are optional — the app falls back to text labels if missing.
#
# Filename                    Used in                   Fallback
# ───────────────────────────────────────────────────────────────────────────

# ── Toolbar ──────────────────────────────────────────────────────────────────
# icons/open.png              Toolbar                   📂 Open
# icons/fit-width.png         Toolbar                   ↔ Fit Width
# icons/fit-page.png          Toolbar                   ⬜ Fit Page
# icons/zoom-in.png           Toolbar                   🔍+
# icons/zoom-out.png          Toolbar                   🔍−
# icons/ocr.png               Toolbar                   🔤 OCR Mode
# icons/fullscreen.png        Toolbar                   [none]


# ── Toolbar — Panel toggles (icon-only, no text fallback) ────────────────────
# icons/panel-thumbnails.png       Toolbar left toggle       (hidden if missing)
# icons/panel-thumbnails-show.png  Toolbar — panel shown state
# icons/panel-thumbnails-hide.png  Toolbar — panel hidden state
# icons/panel-ocr.png              Toolbar right toggle      (hidden if missing)
# icons/panel-ocr-show.png         Toolbar — panel shown state
# icons/panel-ocr-hide.png         Toolbar — panel hidden state

# ── Toolbar — Bookmarks (icon-only, no text fallback) ────────────────────────
# icons/bookmark-off.png      Toolbar bookmark toggle   (page not bookmarked)
# icons/bookmark-on.png       Toolbar bookmark toggle   (page bookmarked)
# icons/bookmarks.png         Toolbar bookmarks list    (open bookmark popup)

# ── Nav bar ──────────────────────────────────────────────────────────────────
# icons/nav-first.png         Nav bar                   ⏮
# icons/nav-prev.png          Nav bar                   ◀ Prev
# icons/nav-next.png          Nav bar                   Next ▶
# icons/nav-last.png          Nav bar                   ⏭
